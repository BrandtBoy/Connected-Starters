Networking hos Connected Starters

Networking hos Connected Starters handler om at møde andre studerende, dele idéer og skabe relationer på tværs af uddannelser.

Gennem vores netværksaktiviteter møder du ligesindede studerende, undervisere og eksterne samarbejdspartnere, som kan inspirere dig og hjælpe dig videre med dine projekter og idéer.

  • Skab relationer på tværs af studieretninger
  • Få nye perspektiver på dine idéer
  • Mød potentielle samarbejdspartnere
  • Udvid dit professionelle netværk
  • Bliv inspireret af andres projekter
